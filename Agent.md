# Agent Guide

Project: Vite + React + Tailwind app for Dutch lessons.

## Commands
- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`
- Preview: `npm run preview`

## Structure
- Entry HTML: `index.html`
- App bootstrap: `src/main.jsx`
- App UI: `src/App.jsx`
- Lesson data: `src/data/lessons.json`
- Global styles: `src/index.css`

## Notes
- Text-to-speech uses the Web Speech API in the browser.
- UI copy is in Spanish and Dutch; keep language consistency when editing.
